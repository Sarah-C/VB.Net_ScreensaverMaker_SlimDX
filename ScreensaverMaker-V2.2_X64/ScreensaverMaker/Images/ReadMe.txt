


To add images to the project:

1: Add the image to the Images folder, by right clicking the folder, and selecting "Add existing..."
2: Open the "My Project" folder.
3: Double click on "Resources.resx"
4: Click on the arrow to the right of the button "Add Resource".
5: Select an image, and it will appear in the list of images in the preview.


It can be accessed with:

Dim image as Bitmap = My.Resources.MyImageName
